Layout

A basic Fairdata layout can be achieved with the header, footer and content components: 


    <body>
        <div class="fd-header container-fluid">
            ...
        </div>
        <div class="fd-content container">
            <p>Page content here</p>
        </div>
        <div class="fd-footer container-fluid">
            ...
        </div>
    </body>

Cookies notification

A notification about cookies can be included to the page. Please refer to Fairdata SSO for including the notification on your page. This repository contains only styles for the notification.
Notification styles are available on dist/notification.css and can be used independently from other Fairdata styles. Notification styles can be updated by running npm run gulp -- sass-notification

Toggle the sample cookies notification:

@@include('../components/_cookie-notification.html')

English:

The Fairdata services use cookies and tracking to ensure security and quality.

View the Fairdata Privacy Policy.
